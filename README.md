# RailsErdMermaid



## Installation

```shell
bundle add rails_erd_mermaid --github="funwarioisii/rails_erd_mermaid"
```

## Usage

```shell
bundle exec erd_mermaid
```

or output to stdout

```shell
bundle exec erd_mermaid --filetype=stdout
```
